# Titanic Survival Prediction

## Objective
Predict whether a passenger survived the Titanic disaster using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Dataset
Titanic Dataset

## Model
Random Forest Classifier

## Accuracy
82%

## Features Used
- Passenger Class
- Gender
- Age
- Siblings/Spouse
- Parents/Children
- Fare
- Embarked

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Feature Encoding
4. Train-Test Split
5. Model Training
6. Prediction
7. Evaluation
